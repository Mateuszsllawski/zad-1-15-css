Cechy modyfikowania formatowania w HTML oraz CSS

HTML CSS
- Aby zmienić sposób wyglądu znaków wyświetlanych na ekranie możemy wykorzystac dwa rodzaje znaczników: style logiczne i fizyczne
- tekst dokumentu HTML jest formatowany przez przeglądarkę wyłącznie na podstawie znaczników. Wszelkie dodatkowe znaki takie jak spacje, tabulacje czy też znaki nowej linii są ignorowane. 		- Możliwość redefiniowania stylów znaczników HTML i tworzenia klas stylów
- Zdefiniowanie zewnętrznego stylu umożliwia zmianę wyglądu całej witryny, poprzez modyfikację jednego pliku

Nowe możliwości CSS w porównaniu z HTML

  1. Możliwość zmiany wyglądu wielu stron naraz bez ingerowania w sam kod
  2. Większe możliwości pozycjonowania elementów na stronie
  3. Więkasza możliwość modyfikacji stylu strony np. Rodzina czcionek, kolor tekstu, marginesy, odstęp międzywierszowy
